# Obsidian插件开发指南

## 🎯 核心要点
- Obsidian插件基于TypeScript开发
- 使用Obsidian API与应用程序交互
- 插件需要manifest.json文件定义基本信息
- 支持自定义设置页面和命令

## 🔧 实际应用
**适用场景**: 需要扩展Obsidian功能时
**使用方法**: 
1. 创建插件目录结构
2. 编写main.ts文件
3. 配置manifest.json
4. 使用npm run build编译
